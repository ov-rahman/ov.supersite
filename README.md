# ⭐ BRAWL HUB

Фан-сайт про обновления и механики **Brawl Stars** в фирменном мультяшном стиле игры.

## Возможности

- 📦 **Список обновлений** (2025–2026: Ramen Rebellion, Сириус и Prestige, Buffies и др.)
  - клик по строке — быстрый просмотр: картинка + краткое описание;
  - клик по **номеру версии** — подробный разбор всех изменений.
- ⚡ Карточки ключевых механик (NanoPowers, Fusions, Prestige, Тени Сириуса…).
- 🤖 **Нейро-ассистент** (кнопка внизу главной):
  - чат с любой нейросетью с OpenAI-совместимым API;
  - ассистент умеет **менять оформление сайта** по команде («сделай в красных тонах» и т.п.);
  - кнопка «↩️ Сброс оформления» возвращает исходный дизайн.
- ⚙️ **Настройки → Нейросеть**: название, URL API, ключ, модель, своя инструкция (системный промпт), сброс истории чата.

Все данные (ключ, история, оформление) хранятся только в браузере (localStorage).

## Запуск

Статический сайт — просто открой `index.html` или подними любой сервер:

```bash
python3 -m http.server 8080
```
